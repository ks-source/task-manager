✅ ステップ1 コマンド（リロード不要版）

⚠️ このバージョンは変数衝突エラーを回避するため、IIFE（即時実行関数）形式になっています。
ページをリロードしなくても実行できますが、最も正確なテストには F5 リロード後の実行を推奨します。
📋 実行手順:
  1. 推奨: ページをリロード（F5キー）←より正確な結果のため
  2. 代替: リロードせずそのまま実行も可能
  3. 下の「📋 コマンドをコピー」ボタンをクリック
  4. 開発者ツールのコンソールに貼り付けて実行（Enter）
  5. すべての出力をコピーして報告
ステップ1: 手動エッジのDOM情報とBBoxを確認（IIFE版）
目的: 手動エッジがDOMに追加されているか、親要素、Line要素のHTML、BBox座標を確認します。
特に重要: BBox の x, y 座標が viewBox 範囲内（0〜1955, 0〜5344）にあるかチェック
(function() { const manualEdges = document.querySelectorAll('[data-et="manual-edge"]'); console.log('手動エッジ数:', manualEdges.length); manualEdges.forEach(edge => { console.log('Edge:', edge.getAttribute('data-id')); console.log(' Parent:', edge.parentElement?.tagName, edge.parentElement?.className); console.log(' Line:', edge.querySelector('line')?.outerHTML); console.log(' Text:', edge.querySelector('text')?.outerHTML); try { const bbox = edge.getBBox(); console.log(' BBox:', bbox.x, bbox.y, bbox.width, bbox.height); } catch(e) { console.log(' BBox: ERROR', e.message); } }); })();
📤 報告してください:
  • 「手動エッジ数:」の値
  • 「Parent:」の値（SVG か他の要素か）
  • 「Line:」のHTML（x1, y1, x2, y2 の座標値）
  • 「BBox:」の値（x, y, width, height）← これが最重要！

期待される BBox の範囲:

  • x: 0 〜 1955 の範囲内
  • y: 0 〜 5344 の範囲内

これらの範囲外にある場合、エッジが画面外に描画されています。
