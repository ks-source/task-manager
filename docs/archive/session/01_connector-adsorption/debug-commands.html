🔍 Phase 2 デバッグコマンド実行ガイド

📋 実行前の準備:
  1. flowchart-mockup-interactive.html を開く
  2. 「✏️ エッジ追加」ボタンをクリック
  3. フローチャート上で2回クリックしてエッジを作成（ラベル: "test" など）
  4. F12 キーで開発者ツールを開き、「Console」タブを選択
  5. 以下のコマンドを順番に実行し、全出力をコピーして報告

デバッグコマンド（順番に実行）

1 DOM上に要素が存在するか確認
目的: 手動エッジがDOMに追加されているか、親要素、BBoxを確認
const manualEdges = document.querySelectorAll('[data-et="manual-edge"]'); console.log('手動エッジ数:', manualEdges.length); manualEdges.forEach(edge => { console.log('Edge:', edge.getAttribute('data-id')); console.log(' Parent:', edge.parentElement?.tagName, edge.parentElement?.className); console.log(' Line:', edge.querySelector('line')?.outerHTML); console.log(' Text:', edge.querySelector('text')?.outerHTML); try { const bbox = edge.getBBox(); console.log(' BBox:', bbox.x, bbox.y, bbox.width, bbox.height); } catch(e) { console.log(' BBox: ERROR', e.message); } });
2 親要素の transform チェーン確認
目的: SVGの viewBox、output グループの transform、親要素チェーンを確認
const svg = document.querySelector('.svg-content-wrapper svg') || document.querySelector('svg'); const output = svg.querySelector('.output') || svg.querySelector('g'); console.log('SVG viewBox:', svg.getAttribute('viewBox')); console.log('output transform:', output?.getAttribute('transform')); if (output) { Array.from(output.children).forEach(child => { console.log(' child:', child.tagName, child.className, 'transform:', child.getAttribute('transform')); }); } const edge = document.querySelector('[data-et="manual-edge"]'); if (edge) { let parent = edge.parentElement; let chain = []; while (parent && parent !== document.body) { chain.push({ tag: parent.tagName, class: parent.className?.baseVal || parent.className, transform: parent.getAttribute('transform'), viewBox: parent.getAttribute('viewBox') }); parent = parent.parentElement; } console.log('Parent chain:', JSON.stringify(chain, null, 2)); }
3 テスト円を追加して表示確認
目的: SVGルート vs output内での表示差を確認
const svg = document.querySelector('.svg-content-wrapper svg') || document.querySelector('svg'); const circle = document.createElementNS('http://www.w3.org/2000/svg', 'circle'); circle.setAttribute('cx', '100'); circle.setAttribute('cy', '100'); circle.setAttribute('r', '50'); circle.setAttribute('fill', 'red'); circle.setAttribute('id', 'test-circle-root'); svg.appendChild(circle); console.log('赤い円をSVGルートに追加 (100,100)'); const output = svg.querySelector('.output > g') || svg.querySelector('.output') || svg.querySelector('g'); if (output) { const circle2 = document.createElementNS('http://www.w3.org/2000/svg', 'circle'); circle2.setAttribute('cx', '100'); circle2.setAttribute('cy', '100'); circle2.setAttribute('r', '50'); circle2.setAttribute('fill', 'blue'); circle2.setAttribute('id', 'test-circle-output'); output.appendChild(circle2); console.log('青い円をoutput内に追加 (100,100)'); }
⚠️ 視覚的確認: 赤い円・青い円がどう見えるか報告してください
（見えない / 見える / 位置が違う / 画面外 など）
4 既存ノードと同じ親に追加
目的: PHASE1ノードと同じ親に要素を追加して表示確認
const phase1 = document.querySelector('[data-id="PHASE1"]'); if (phase1) { const parent = phase1.parentElement; console.log('PHASE1 parent:', parent.tagName, parent.className); const circle = document.createElementNS('http://www.w3.org/2000/svg', 'circle'); circle.setAttribute('cx', '1600'); circle.setAttribute('cy', '848'); circle.setAttribute('r', '30'); circle.setAttribute('fill', 'lime'); circle.setAttribute('stroke', 'black'); circle.setAttribute('stroke-width', '3'); parent.appendChild(circle); console.log('緑の円をPHASE1と同じ親に追加 (1600,848)'); }
⚠️ 視覚的確認: 緑の円が見えるか、PHASE1ノードの上に表示されているか報告してください
