🔄 ステップ1 再実行コマンド

⚠️ 必須: このコマンドを実行する前に、ページをリロード（F5）してください！
📋 実行手順:
  1. ページをリロード（F5キー）で新しい状態にする
  2. 「✏️ エッジ追加」ボタンをクリック
  3. フローチャート上で2回クリックしてエッジを作成
  4. ラベルを入力（例: "test"）
  5. F12 で開発者ツールを開き、「Console」タブを選択
  6. 下の「📋 コマンドをコピー」ボタンをクリック
  7. コンソールに貼り付けて実行（Enter）
  8. すべての出力をコピーして報告
ステップ1: 手動エッジのDOM情報とBBoxを確認
目的: 手動エッジがDOMに追加されているか、親要素、Line要素のHTML、BBox座標を確認します。
特に重要: BBox の x, y 座標が viewBox 範囲内（0〜1955, 0〜5344）にあるかチェック
const manualEdges = document.querySelectorAll('[data-et="manual-edge"]'); console.log('手動エッジ数:', manualEdges.length); manualEdges.forEach(edge => { console.log('Edge:', edge.getAttribute('data-id')); console.log(' Parent:', edge.parentElement?.tagName, edge.parentElement?.className); console.log(' Line:', edge.querySelector('line')?.outerHTML); console.log(' Text:', edge.querySelector('text')?.outerHTML); try { const bbox = edge.getBBox(); console.log(' BBox:', bbox.x, bbox.y, bbox.width, bbox.height); } catch(e) { console.log(' BBox: ERROR', e.message); } });
📤 報告してください:
  • 「手動エッジ数:」の値
  • 「Parent:」の値（SVG か他の要素か）
  • 「Line:」のHTML（x1, y1, x2, y2 の座標値）
  • 「BBox:」の値（x, y, width, height）← これが最重要！
